# Pixel Runner — Blast & Collect

A pixel-art infinite runner built with Next.js 14 App Router, HTML5 Canvas, and the Web Audio API.

## 🎮 Gameplay

- **Run, jump, and blast** through 3 themed zones: Forest, Dungeon, and Lava
- **Find the golden box** at the end of each level to collect a letter
- After all 3 boxes are found, the **Final Chamber** unlocks
- Place all 3 golden boxes on the pedestal to reveal the win message

## 🕹️ Controls

| Action | Keys |
|--------|------|
| Move   | Arrow Keys or WASD |
| Jump   | ↑ / W / Z |
| Shoot  | Space / Click |

## 🚀 Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/pixel-runner-game
cd pixel-runner-game
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## ☁️ Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project** → Import your repo
3. Vercel auto-detects Next.js — just click **Deploy**
4. No environment variables needed
